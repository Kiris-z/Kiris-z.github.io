相联存储器 (CAM)：可以按内容访问的存储器。可以按照内存检索到存储位置进行读写，“快表 TLB” 是一种相联存储器

ROM 和 RAM 指的都是半导体存储器。

CD-ROM 不是随机存取器，而是串行存储器

# 存储器的性能指标

存取时间不等于存取周期，通常存取周期大于存取时间。这是因为对任何一种存储器，在读写操作之后，都要有一段恢复内存状态的复原时间。对于破坏性读出的存储器，存取周期往往比存取时间大得多，因为存储器中的信息读出后需要马上进行再生。

# 多级存储系统

主存和 Cache 之间的数据调动是由硬件自动完成，对所有程序员均是透明的；

主存和辅存之间的数据调动则是由硬件和操作系统共同完成，对应用程序员是透明的

# 刷新

刷新的单位是行

刷新周期：对 DRAM 的所有存储单元恢复一次原状态的时间间隔

刷新间隔：两次刷新的起始时间差

行刷新间隔：两行刷新的起始时间差

刷新时间：规定的一个周期内刷新的总时间

# 高位交叉编址

不满足程序局部性原理：由于高位多体交叉存储器（或称连续编址存储器）在一个存储体内地址连续，所以一个程序的指令和数据极大可能只分布在一个存储体上，这样的只能顺序读取，而无法充分利用并行存取。（即一次并行取出来的指令或数据不在一个程序的局部上，跨度很大，可能是在一个程序的首尾，或不同的程序，也就不再满足局部性原理。（事实上，这里仅作解释使用，准确而言，高位多体交叉只能顺序存取），所以无法提高 CPU 访问存储器的速度。

# 双端口存储器

双端口存储器由于同一个存储器具有两组相互独立的读写控制电路而得名。由于进行并行的独立操作，因而是一种高速工作的存储器

当两个端口的地址不相同时，在两个端口上进行读写操作，一定不会发生冲突。可以同时访问同一区间，同一单元。

当两个端口同时存取存储器同一存储单元时，便发生读写冲突，同时读不冲突，一读一写或同时写冲突

# 闪存

ROM 的一种，是E2PROME^2PROME2PROM 的进一步发展，可读可写

读速度比写快很多，因为写时需要擦除原有数据；是一种半导体存储器；非易失性存储器；随机访问方式

常见的是 SSD 固态硬盘，其由 Flash 芯片组成

# 注意

题目中，交叉编址方式指的是低位交叉编址
