# 已知先序序列求不同二叉树个数

先序序列为 a0a1a2⋯ana_0a_1a_2\cdots a_na0​a1​a2​⋯an​ ，长度为 n+1n+1n+1 的先序序列不同二叉树个数为 cn+1c_{n+1}cn+1​。

已知先序序列和中序序列可以唯一确定一个二叉树，考虑中序序列。

中序序列类似于 b0⋯bk−1a0bk⋯bn−1b_0\cdots b_{k-1} a_0 b_{k} \cdots b_{n-1}b0​⋯bk−1​a0​bk​⋯bn−1​。

b0⋯bk−1b_0\cdots b_{k-1}b0​⋯bk−1​ 为左子树的节点 同时 a1⋯aka_1\cdots a_{k}a1​⋯ak​ 也为左子树节点，二者应为同一组数的不同排列，且已知先序序列，转化为原问题的子问题，其个数为 ckc_{k}ck​，同时右子树的个数为 cn−kc_{n-k}cn−k​，于是根据 a0a_0a0​ 在中序序列中所处的不同位置可得

cn+1=∑k=0nck⋅cn−kc_{n+1} = \sum_{k=0}^{n}c_k\cdot c_{n-k}cn+1​=k=0∑n​ck​⋅cn−k​

此为卡特兰 (Catalan) 数递推公式，故

cn=C2nn−C2nn−1=1n+1C2nnc_n = C_{2n}^n-C_{2n}^{n-1} =\frac{1}{n+1}C_{2n}^ncn​=C2nn​−C2nn−1​=n+11​C2nn​

# 前驱寻找

  1. 根节点 无前驱
  2. 节点为双亲左孩子，或为双亲右孩子且其双亲没有左子树，双亲即为前驱
  3. 节点为双亲右孩子且其双亲有左子树，左子树最右节点即为前驱

# 后继寻找

  1. 有左子树
    左孩子即为后继
  2. 无左子树但有右子树
    右孩子即为后继
  3. 无左子树也无右子树（叶节点）
    右链域即为后继

# 中序线索二叉树

# 前驱寻找

  1. 无左子树
    左链域即为前驱
  2. 有左子树
    左子树最右节点即为前驱

# 后继寻找

  1. 无右子树
    右链域即为后继
  2. 有右子树
    右子树最左节点即为后继

# 后序线索二叉树

# 前驱寻找

  1. 无左子树
    左链域即为前驱
  2. 有左子树但无右子树
    左孩子即为前驱
  3. 有左子树也有右子树
    右孩子即为前驱

# 后继寻找

  1. 根节点 无后继
  2. 节点为双亲右孩子，或其双亲的左孩子且双亲没有右子树 双亲即为后继
  3. 节点为双亲左孩子且其双亲有右子树 右子树最左节点即为后继

# 注意

其中先序线索二叉树找前驱和后序线索二叉树找后继均需要知道双亲节点，需要使用带标志域的三叉链表作为存储结构。
