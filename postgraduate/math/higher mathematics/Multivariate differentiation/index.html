# 全微分

  1. 全微分条件存在 AAA, BBB 使得 Δz=AΔx+BΔy+o((Δx)2+(Δy)2)\Delta z = A\Delta x+B\Delta y+o(\sqrt{(\Delta x)^2+(\Delta y)^2})Δz=AΔx+BΔy+o((Δx)2+(Δy)2​) 成立，注意 A，B 的修饰词是存在，也就是说，是现有 A，B，然后再通过计算得出了 A,B 的值和偏导值相同
  2. 偏导数连续可以推出可微，注意偏导数依然是一个二元函数，数学表达式为lim⁡(x,y)→(0,0)fx′(x,y)=fx′(0,0)\lim_{(x,y)\rightarrow (0,0)}f'_x(x,y)=f'_x(0,0)lim(x,y)→(0,0)​fx′​(x,y)=fx′​(0,0), 可微对应切平面，偏导连续对应切平面由共面的各连续偏导构成。
  3. “两个线属性存在，并不能推出一个可微的面属性”

# 隐函数求导

对于 F(x,u(x))=0F(x,u(x)) = 0F(x,u(x))=0，我们可以通过 dudx=−Fx′Fy′\frac{du}{dx} = -\frac{F_x'}{F_y'}dxdu​=−Fy′​Fx′​​ 求出，但对于 F(x,u(x),v(x))=0F(x,u(x),v(x)) = 0F(x,u(x),v(x))=0，我们则不能用如上的方式求，可以通过探求 dudx=−Fx′Fy′\frac{du}{dx} = -\frac{F_x'}{F_y'}dxdu​=−Fy′​Fx′​​ 该公式的来源来理清，如下

对于F(x,u(x))=0Fx′+Fu′dudx=0⇒dudx=−Fx′Fy′而对于F(x,u(x),v(x))=0Fx′+Fu′dudx+Fv′dudx=0\text{对于}\ F(x,u(x)) = 0\\ F_x'+F_u'\frac{du}{dx} = 0\Rightarrow\frac{du}{dx} = -\frac{F_x'}{F_y'}\\ \text{而对于}\ F(x,u(x),v(x))=0\\ F_x'+F_u'\frac{du}{dx}+F_v'\frac{du}{dx}=0对于 F(x,u(x))=0Fx′​+Fu′​dxdu​=0⇒dxdu​=−Fy′​Fx′​​而对于 F(x,u(x),v(x))=0Fx′​+Fu′​dxdu​+Fv′​dxdu​=0

可以看出，由于 v(x)v(x)v(x) 的干扰，无法得出如上公式。
故运用隐函数求导法则时，需要保证 F 的自变量中，只有 x 以及 u (x) 两个 x 的相关项，其余自变量均不能为 x 的函数。
