  1. 极限 $\lim_{x\rightarrow +\infty}\frac{(1+\frac{1}{x})^{x^2}}{e^x}=\lim_{x\rightarrow +\infty}\frac{e^x}{e^x}=1$。 {.quiz .false}

  2. 设 $f(x)$ 在 $x=a$ 的某个邻域内有定义，下面哪个选项不能说明x=a处可导： {.quiz .multi}

    • $\lim_{h\rightarrow +\infty}h[f(a+\frac{1}{h})-f(a)]$ 存在 {.correct}
    • $\lim_{h\rightarrow 0}\frac{f(a+2h)-f(a+h)}{h}$ 存在 {.correct}
    • $\lim_{h\rightarrow 0}\frac{f(a+h)-f(a-h)}{2h}$ 存在 {.correct}
    • $\lim_{h\rightarrow 0}\frac{f(a)-f(a-h)}{h}$ 存在
    • $\lim_{n\rightarrow \infty}n[f(a+\frac{1}{n})-f(a)]$ 存在 {.correct}

    {.options}

    • ：单侧极限
    • ：极限之和存在不能说明分别存在,反例$f(x)=0(x\neq 0),f(0)=1$
    • ：极限之和存在不能说明分别存在，反例$f(x)=|x|$
    • ：正确
    • : n趋于无穷意思为n趋于正无穷，单侧极限

    {.options}

    https://www.oihere.com/zh-cn/thread/4d295afa-31e4-4086-9f5f-030857bbb6b1/?order=all
    https://www.yulucn.com/question/184251461

  3. 求极限$\lim_{x \rightarrow 0} \frac{\int_0^x \sin|\frac{1}{t}|dt}{x}$，答案为 [$\frac{2}{\pi}$]{.gap}。{.quiz .fill}

    洛必达法则运用条件，洛后极限不存在时不能使用。
    若不加绝对值，极限为0。

    [https://www.zhihu.com/question/437823705]{.mistake}
    [https://www.zhihu.com/question/437521540]{.mistake}

  4. 设 $\cos x-1=x\sin\alpha (x)$，其中 $|\alpha (x)| < \frac{\pi}{2}$,则当 $x\rightarrow 0$ 时,$\alpha (x)$ 是[]{.gap} 。 {.quiz}

    • 比 x 高阶的无穷小量
    • 比 x 低阶的无穷小量
    • 与 x 同阶但不等价的无穷小量 {.correct}
    • 与 x 等价的无穷小量

    {.options}

    $\lim_{x\rightarrow 0}\frac{\sin\alpha (x)}{x}=\lim_{x\rightarrow 0}\frac{\cos x-1}{x^2}=\lim_{x\rightarrow 0}\frac{-\frac{1}{2}x^2}{x^2}=-\frac{1}{2}$，
    即有 $\lim_{x\rightarrow 0} \sin \alpha (x) = 0$，又 $|\alpha (x)| < \frac{\pi}{2}$，则 $\lim_{x\rightarrow 0} \alpha (x) = 0$。
    $\lim_{x\rightarrow 0}\frac{\alpha (x)}{x} = \lim_{x\rightarrow 0}\frac{\sin \alpha(x)}{x} = -\frac{1}{2}$。

    注意等价无穷小的使用条件

  5. 求极限$\lim_{x\rightarrow 1}(1-x^2)\tan \frac{\pi}{2}$，答案为 [$\frac{4}{\pi}$]{.gap}。{.quiz .fill}

    注意x趋向1，不是0

  6. 设 $f(x)$ 二阶可导，$f(0)=0,f’(0)=1,f’’(0)=2$ ，求极限$\lim_{x\rightarrow 0}\frac{f(x)-x}{x^2}$，答案为 [1]{.gap}。{.quiz .fill}

    原式 = $\lim_{x\rightarrow 0}\frac{f’(x)-1}{2x}=\lim_{x\rightarrow 0}\frac{f’’(x)}{2}=\frac{f’’(0)}{2}=1$ 错误，二阶可导不能推出二阶导函数极限存在，也不能推出二阶导函数连续。

  7. 求极限 $\lim_{x\rightarrow +\infty}\ln(1+e^x)-x$ ,答案为 [0]{.gap}。{.quiz .fill}

    等价变换 $x = \ln e^x$ 。

  8. 极限 $\lim_{x\rightarrow 0}\frac{\cos 2x - \cos x}{x^2}$ 可以使用拉格朗日吗，为什么? {.quiz .false}

    2x与x不为等价无穷小，不能使用拉格朗日
    当 $f’(\xi)!=0$，则可以使用拉格朗日
    当 $f’(\xi)=0，需要判断函数内部是否等价$ ，本质上是 $\xi$ 趋近0的速度与 $x$ 相比无法确定

  9. 极限 $\lim_{x\rightarrow 0,y\rightarrow 0}\frac{xy^2}{x^2+y^2}$，答案为[0]{.gap}。{.quiz .fill}

    绝对值极限趋向零，则极限趋向b 0

  10. 数列极限 $\lim_{n\rightarrow \infty} x_n$ 存在 $\Leftrightarrow \lim_{n\rightarrow \infty}\frac{x_{n+1}}{x_n}=1$。{.quiz .false}

    注意等价符号，兼顾充分性与必要性

  11. 若存在 $\delta >0$，当$0<|x-x_0|<\delta$时，$f(x)>g(x)\Rightarrow \lim_{x\rightarrow x_0}f(x)\ge \lim_{x\rightarrow x_0}g(x)$。{.quiz .false}

    未假设极限存在

  12. 数列极限$\lim_{n\rightarrow \infty}\frac{(\ln n)’}{n’}=\lim_{n\rightarrow}\frac{1}{n}=0$。{.quiz .false}

    n为正整数，对数列没有导数的概念。

  13. 极限$(1+x)^{x^2}-1$为几阶无穷小,[3]{.gap}。{.quiz .fill}

    无穷小的逆用，$(1+x)^{x^2}-1\sim \ln[(1+x)^{x^2}-1+1]\sim x^2\ln(1+x)\sim x^3$

  14. 设$\lim_{x\rightarrow x_0^+}f’(x)=\lim_{x\rightarrow x_0^-}f’(x)=a$，则$f(x)\text{在}x=x_0\text{处必可导且}f’(x_0)=a$。{.quiz .false}

    区分$f(x)$在$x=x_0$的左右导数$f’-(x_0)$与$f’+(x_0)$与$f’(x)\text{在}x=x_0$处的左右极限$\lim_{x\rightarrow x_0^-}f’(x)$与$\lim_{x\rightarrow x_0^+}f’(x)$
